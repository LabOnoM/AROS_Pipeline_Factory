# Workspace Management Context

## 🧭 Domain Context
This pipeline houses the global workflows, project structure templates, and onboarding scripts that apply to the entire repository and the live AROS runtime.

## ⚖️ Component Rules
- **Wiki Integration**: Handles `/wiki-ingest`, `/wiki-query`, `/wiki-research`, and `/wiki-update` workflows.
- **Version Control**: Handles the `regent-governance` skill and the `/lab-commit` and `/lab-reorganize` canonical workflows.
- Any modifications to workflows here usually affect ALL pipelines, thus falling under SAMS compliance. Consult `01.Shared_Assets` governance rules.

## 🚀 Execution
- **Trigger**: Workflows like `/science-project-onboarding` and `/audit-shared-assets` are managed here.
