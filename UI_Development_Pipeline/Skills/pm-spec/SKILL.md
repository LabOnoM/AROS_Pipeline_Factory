【模板: PRD / Product Spec】
【意图】产品需求文档单页, 结构清晰。
【布局】
- Title bar + 状态 pill (draft/approved/shipped)
- Problem & Why now
- Success metrics (3-5 个 KPI)
- Scope (in / out)
- User stories (Given/When/Then)
- Design notes (含占位 mockup)
- Rollout plan + Open questions
