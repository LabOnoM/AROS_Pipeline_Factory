# UI Development Pipeline Context

## 🧭 Domain Context
This pipeline holds skills and tools for frontend web interface development, UI/UX design via A2UI protocols, and browser automation.

## ⚖️ Component Rules
- **Design Principles**: Must adhere to Modern Web Guidance (Glassmorphism, view transitions, high-performance LCP).
- **Frameworks**: Standardly uses React/Next.js/Vite with robust responsive container queries.
- **Wiki Pointers**: Check `.wiki/features/ui-ux-features.md` (if existing) before undertaking major interface modifications.
