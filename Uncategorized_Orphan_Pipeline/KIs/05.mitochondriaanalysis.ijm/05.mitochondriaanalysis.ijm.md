# ImageJ Macro: Batch Mitochondria Morphology Analysis

This Knowledge Item contains an ImageJ macro script (`.ijm`) designed to automate and batch-process the analysis of mitochondrial morphology from time-lapse images of single cells.

## Key Features

- **Batch Processing**: Automates analysis for an entire directory of images.
- **User-Friendly**: Prompts the user for input and output directories at runtime.
- **Customizable**: The file suffix (`.tif`, `.png`, etc.) can be easily changed in the initial dialog.
- **Dependency-Aware**: Checks for the required ImageJ version.

## Dependencies

- **ImageJ**: Version `1.53t` or newer.
- **Plugin**: The **'Mitochondria Analyzer'** plugin must be installed in ImageJ. This macro is a wrapper script for that plugin and will not function without it.

## Usage

1.  **Installation**:
    -   Save the macro code below as `05.mitochondriaanalysis.ijm` in your ImageJ `macros` folder.
    -   Install the macro set by navigating to `Plugins > Macros > Install...` in ImageJ and selecting the saved file.
2.  **Execution**:
    -   Run the macro from `Plugins > Macros > 05.mitochondriaanalysis.ijm`.
    -   A dialog box will appear.
    -   **Input directory**: Select the folder containing your single-cell time-lapse images.
    -   **Output directory**: Select the folder where the analysis results should be saved.
    -   **File suffix**: Specify the file extension of your images (e.g., `.tif`, `.czi`).
    -   Click **OK** to start the batch analysis.

## Macro Code (`05.mitochondriaanalysis.ijm`)

```ijm
// Macro to batch analyze mitochondria morphology from time-lapse images
//
// This macro requires the 'Mitochondria Analyzer' plugin to be installed.
//
// It will:
// 1. Ask the user for an input directory of images.
// 2. Ask the user for an output directory to save results.
// 3. Process each image in the input directory that matches the given suffix.
// 4. For each image, it runs the Mitochondria Analyzer plugin.
// 5. Saves the results to the specified output directory.

#@ File (label = "Input directory", style = "directory") input
#@ File (label = "Output directory", style = "directory") output
#@ String (label = "File suffix", value = ".tif") suffix

function processFolder(input, output, suffix) {
    list = getFileList(input);
    print("Processing " + list.length + " files in input directory...");
    for (i = 0; i < list.length; i++) {
        if(endsWith(list[i], suffix)) {
            print("Processing file: " + list[i]);
            processImage(input, output, list[i]);
        }
    }
}

function processImage(input, output, filename) {
    open(input + File.separator + filename);
    imageTitle = getTitle();

    // NOTE: The 'run' command below is a placeholder.
    // The exact command and its parameters depend on the 'Mitochondria Analyzer' plugin.
    // You may need to record the macro command once to get the precise syntax.
    // Example: run("Mitochondria Analyzer", "parameter1=value1 parameter2=value2");
    print("Running Mitochondria Analyzer on " + imageTitle);
    run("Mitochondria Analyzer", "input=" + imageTitle + " output=" + output + " show=Nothing");

    // This section assumes the plugin creates a 'Results' table.
    // If the plugin saves files automatically, this may not be needed.
    if (isOpen("Results")) {
       saveAs("Results", output + File.separator + replace(imageTitle, suffix, "") + "_results.csv");
       close("Results");
    }


    close("*"); // Close all image windows to conserve memory
}

// --- Main Execution Block ---
requires("1.53t"); // Sets the minimum required ImageJ version.

print("\\Clear"); // Clears the log window
print("Batch Mitochondria Analysis Macro -- Started");
print("Input Directory: " + input);
print("Output Directory: " + output);

setBatchMode(true); // Prevents image windows from popping up, speeding up the process
processFolder(input, output, suffix);
setBatchMode(false);

print("Batch processing complete.");

```
