# Global Policy: Contextual Relevance Check

This document specifies the Contextual Relevance Check policy.

## Workflow

This section details the step-by-step workflow an agent must follow.
1. Intercept User Request.
2. Analyze Context.
3. Determine Relevance.
4. Check Environment.
5. Decision Point.
