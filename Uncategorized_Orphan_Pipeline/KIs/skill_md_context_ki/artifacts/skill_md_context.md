# SKILL.md Contextual Definition

This Knowledge Item clarifies the nature and usage of the `SKILL.md` file within the AROS ecosystem.

## Generic Template
The file `SKILL.md` is a generic template name used to define the rules, documentation, and metadata for individual skills. It is not a unique entity in itself but rather a placeholder name for skill definitions.

## Contextual Mappings
Each instance of `SKILL.md` is contextualized by its parent directory, which represents the unique skill it defines.

- **q-and-a-prep-partner:** The `SKILL.md` located at `/home/owner03/.gemini/skills/q-and-a-prep-partner/SKILL.md` defines the 'q-and-a-prep-partner' skill.
- **dei-statement-drafter:** The `SKILL.md` located at `/home/owner03/.gemini/skills/dei-statement-drafter/SKILL.md` defines the 'dei-statement-drafter' skill.
- **anatomy-quiz-master:** The `SKILL.md` located at `/home/owner03/.gemini/skills/anatomy-quiz_master/SKILL.md` defines the 'anatomy-quiz-master' skill.

These mappings ensure that while `SKILL.md` is a common filename, its specific content and meaning are always tied to the unique skill directory it resides within.
