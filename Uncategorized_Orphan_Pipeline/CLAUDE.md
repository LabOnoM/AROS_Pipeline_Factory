# Uncategorized Orphan Pipeline Context

## 🧭 Domain Context
This is a catch-all pipeline for unsorted system assets and external skills that have not yet been assigned a permanent domain.

## ⚖️ Component Rules
- **Refactoring Goal**: Assets here should ideally be categorized, refactored, and moved to their proper domain pipeline.
- **Do Not Share**: Avoid referencing skills in this pipeline from Global Workflows until they are promoted to `01.Shared_Assets/` or a stable domain pipeline.
