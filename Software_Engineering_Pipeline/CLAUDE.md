# Software Engineering Pipeline Context

## 🧭 Domain Context
This pipeline contains the code generation, test orchestration, and validation tools used by AROS software agents.

## ⚖️ Component Rules
- **Quality Gates**: All generated software must pass through the local testing protocols.
- **Integrity**: Enforces the AROS GEPA (Generative Evolutionary Process Architecture) guidelines for writing self-healing and robust code.

## 🚀 Execution
- **Trigger**: `/qa-system-audit` governs global QA and software integrity tests.
