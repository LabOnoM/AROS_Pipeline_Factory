# Marker Gene Reference for MSC and Bone Marrow Stroma

This reference lists the marker genes used for cell type scoring and annotation in mouse bone marrow vs. cultured MSC comparative analyses.

## MSC and Stromal Lineages

| Category | Genes |
|----------|-------|
| **MSC / Stromal** | `Lepr`, `Cxcl12`, `Kitl`, `Pdgfra`, `Pdgfrb`, `Vcam1`, `Nt5e`, `Eng`, `Thy1` |
| **Osteogenic** | `Sp7`, `Bglap`, `Col1a1`, `Runx2`, `Alpl`, `Ibsp`, `Spp1` |
| **Adipogenic** | `Adipoq`, `Pparg`, `Fabp4`, `Cebpa`, `Lpl`, `Plin1` |
| **Chondrogenic** | `Sox9`, `Col2a1`, `Acan`, `Col11a1` |

## Niche-Resident Populations

| Population | Genes |
|------------|-------|
| **Endothelial** | `Pecam1`, `Cdh5`, `Kdr`, `Emcn` |
| **Smooth Muscle / Perivascular** | `Acta2`, `Myh11`, `Tagln`, `Des` |
| **Erythroid** | `Hba-a1`, `Hba-a2`, `Hbb-bs`, `Hbb-bt` |
| **Immune (General)** | `Ptprc`, `Cd68`, `Cd3e`, `Cd19`, `Nkg7` |

## Culture Adaptation and Stress

| Category | Genes |
|----------|-------|
| **Culture Stress** | `Hspa1a`, `Hspa1b`, `Hsp90aa1`, `Hsph1`, `Jun`, `Fos`, `Atf3` |
| **ECM Remodeling** | `Mmp2`, `Mmp9`, `Mmp14`, `Timp1`, `Timp2`, `Fn1`, `Col3a1` |
| **Proliferation** | `Mki67`, `Top2a`, `Pcna`, `Cdk1`, `Ccnb1`, `Ccna2` |
| **Cytokines** | `Il6`, `Csf1`, `Vegfa`, `Tgfb1`, `Bmp2`, `Igf1` |

## Key Comparative Biomarkers

- **Bone Marrow Niche**: *Cxcl12*, *Lepr*, *Kitl*.
- **Cultured / Expanded MSCs**: *Actb*, *Timp1*, *Fn1*, *S100a6*, *Lgals1*.
- **Hematopoietic Contaminants (BM only)**: *Ptprc* (CD45), *Hba-a1*.
