# Request Intent Validation

This skill ensures that before any content retrieval or generation is initiated, the system must accurately interpret and validate the user's core intent and explicit keywords from their request.

## Rule: Intent and Keyword Validation
Before initiating content retrieval or generation, the system must accurately interpret and validate the user's core intent and explicit keywords from their request. This prevents misinterpretations and ensures alignment with the user's explicit needs.
