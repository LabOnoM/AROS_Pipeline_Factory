# Bioinformatics Pipeline Context

## 🧭 Domain Context
This pipeline manages genomic, proteomic analysis, sequence alignments, and structural modeling.

## ⚖️ Component Rules
- **Data Repositories**: Grounded in major scientific databases (e.g., STRING DB, Ensembl, ClinVar).
- **Tooling**: Heavy reliance on R scripts, Python bioinformatics libraries, and API calls to remote sequence databases.

## 🚀 Execution
- **Trigger**: Relevant workflows include `/proteomics-enrichment`.
