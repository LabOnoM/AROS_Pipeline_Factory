# Writing & Publishing Pipeline Context

## 🧭 Domain Context
This pipeline focuses on scholarly publication formats beyond standard manuscripts, such as academic posters, LaTeX templates, and citation formatting engines.

## ⚖️ Component Rules
- **Literature Integration**: Tools here heavily leverage the `literature-ingestion` framework for converting PDFs into markdown and extracting structural JSON metadata.
