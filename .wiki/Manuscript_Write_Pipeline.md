# Manuscript Write Pipeline

The Manuscript Write Pipeline is a dual-agent autonomous system for authoring scientific papers.

## Key Features
- **Markdown-First Authoring**: Strictly prohibits direct LaTeX editing to prevent truncation errors.
- **Data Extraction**: Automated extraction of quantitative data from experimental artifacts.
- **Peer-Review Evaluation**: Rigorous internal assessment before submission.
