# Skill: Excel Injection

Standardized mechanism for injecting markdown narratives into formal Excel application forms (e.g., MEXT SPReAD).

## Components
- `generic_inject_excel.py`: Parametersized script that maps project markdown files to Excel cell addresses defined in the funder profile.
