# Skill: Secure HTML Delivery

Standardized mechanism for generating AES-256 encrypted interactive HTML reports.

## Components
- `generic_interactive_report.py`: Combines markdown sections into a single-file HTML report.
- `build_secure_grant.py`: Encrypts the HTML report using AES-256-GCM.

## Usage
Refer to the `SKILL.md` in `~/.gemini/antigravity/skills/secure-html-delivery/`.
