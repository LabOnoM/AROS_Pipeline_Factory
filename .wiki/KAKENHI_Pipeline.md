# KAKENHI Pipeline

The KAKENHI Pipeline is a specialized workflow for managing JSPS KAKENHI grant reports and applications.

## Key Features
- **Annual Report Generation**: Automates the creation of F-7-2 and other JSPS forms.
- **Figure Integration**: Automated processing and embedding of experimental figures.
- **Dual-Agent Review**: Simulated peer review and compliance checks.
